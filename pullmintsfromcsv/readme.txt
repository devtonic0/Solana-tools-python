this program allows you to pull mint transactions from a csv, its a simple program, set the mint cost you are targeting, pull your transaction csv from solscan, name it transaction.csv and put it in this folder. 

set your price points in find mints.py, the sender address seen in there is a filter to remove that wallet from the list, if you have any wallets you want excluded remove them. after this is done run sortmints.py to clean up the data
